# InAmigos Education

InAmigos Education is an AI-generated education website created for **Task 3: AI Website Generation** during the InAmigos Foundation internship.

## Live Website

Visit the live website here:

[https://inamigos-edu.netlify.app](https://inamigos-edu.netlify.app)

## Project Objective

The objective of this task was to explore AI-powered website creation and understand how AI tools can help generate a complete website structure for any chosen domain.

## Domain

**Education**

The website represents an education-focused platform that helps students and working professionals learn practical skills through guided courses, mentoring, and project-based learning.

## Website Sections

- Home
- About
- Services
- Courses
- Learner Feedback
- Contact

## Features

- Responsive website layout
- Clean navigation menu
- AI-generated hero image
- Education-focused content
- Course and service sections
- Contact enquiry form demo
- Mobile-friendly design

## Tech Stack

- HTML5
- CSS3
- JavaScript

## Files Included

- `index.html` - Main website structure
- `styles.css` - Website styling and responsive design
- `script.js` - Navigation and form interaction
- `assets/learning-hero.png` - Hero image asset

## Internship Task Details

**Task 3: AI Website Generation**

Create a complete website using AI tools for any domain such as business, NGO, portfolio, or education. The website should include multiple sections like homepage, about, and services. The focus is on structure and idea rather than perfection.
